## Available Scripts

```bash
# install dependencies
npm install

# development environment
npm run dev
```

## Deployment

To deploy:

```bash
# build the app
npm run build

# deploy to a unique preview URL
# ntl deploy

# deploy the site into production
# ntl deploy --prod
```
